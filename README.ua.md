## Про проект

**CoffeeJoy** — це командний навчальний frontend-проект у форматі landing page для кав’ярні. Сайт знайомить користувачів із концепцією бренду, перевагами кав’ярні, атмосферою закладу, відгуками клієнтів та контактами.

Користувачі можуть:

- Дізнатися більше про кав’ярню та її філософію.
- Переглянути переваги сервісу та атмосферні фото.
- Ознайомитися з відгуками відвідувачів.
- Знайти контактну інформацію та адресу кав’ярні.
- Підписатися на оновлення через форму.

Мета проекту — створити стильний, адаптивний та зручний сайт, який передає атмосферу кав’ярні та допомагає користувачу швидко знайти потрібну інформацію.

## Моя роль у команді

У цьому командному проекті я виконував роль **Team Lead** та відповідав за секції **Location** і **Coffee Moments**.

Мій внесок:

- Координація роботи команди та організація процесу розробки.
- Розробка секції **Coffee Moments** з адаптивною галереєю зображень.
- Розробка секції **Location** з контактною інформацією, адресою та посиланням на маршрут.
- Адаптивна верстка для mobile, tablet та desktop.
- Робота зі структурою partials, стилями та компонентним підходом у Vite-проекті.
- Підтримка єдиного візуального стилю сайту та узгодження секцій із загальним дизайном.

## ✨ Функціонал

- **Адаптивний landing page:** коректне відображення на mobile, tablet та desktop.
- **Навігація по секціях:** швидкий перехід до About us, Benefits, Gallery, Testimonial та Location.
- **Coffee Moments:** атмосферна галерея зображень, яка показує стиль і настрій кав’ярні.
- **Location:** контактний блок з email, телефоном, адресою та посиланням Get Directions.
- **Benefits:** презентація основних переваг кав’ярні.
- **Testimonials:** блок з відгуками відвідувачів.
- **Subscription form:** форма підписки з модальним вікном підтвердження.
- **Mobile menu:** окреме меню для мобільної версії.
- **UI-анімації:** hover-ефекти, анімація заголовка вкладки та плавна взаємодія з елементами.

## Технологічний стек

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Bundler:** [Vite](https://vitejs.dev/)
- **Деплой:** GitHub Pages
- **Бібліотеки та інструменти:**
  - [Swiper](https://swiperjs.com/) — слайдери та інтерактивні блоки
  - [body-scroll-lock](https://github.com/willmcpo/body-scroll-lock) — керування прокруткою при відкритих меню або модальних вікнах
  - [skroll](https://www.npmjs.com/package/skroll) — анімації під час скролу
  - [gh-pages](https://www.npmjs.com/package/gh-pages) — деплой проекту на GitHub Pages

## Встановлення та запуск

### Передумови

- Встановлений [Node.js](https://nodejs.org/) актуальної LTS-версії.

### Кроки

1. **Клонуйте репозиторій:**

   ```bash
   git clone <repository-url>
   ```

2. **Встановіть залежності:**

   ```bash
   npm install
   ```

3. **Запустіть проект у режимі розробки:**

   ```bash
   npm run dev
   ```

4. **Відкрийте посилання, яке з’явиться у терміналі**  
   зазвичай це `http://localhost:5173/`.

### Збірка проекту

```bash
npm run build
```

### Перегляд production-збірки

```bash
npm run preview
```
