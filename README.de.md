## Über das Projekt

**CoffeeJoy** ist ein teamorientiertes Lernprojekt im Frontend-Bereich in Form einer Landing Page für ein Café. Die Website stellt das Markenkonzept, die Vorteile des Cafés, die visuelle Atmosphäre, Kundenbewertungen und Standortinformationen vor.

Nutzer können:

- Mehr über das Café und seine Philosophie erfahren.
- Die wichtigsten Servicevorteile und die Bildergalerie ansehen.
- Kundenbewertungen lesen.
- Kontaktinformationen und die Adresse des Cafés finden.
- Sich über ein Formular für Neuigkeiten anmelden.

Ziel des Projekts ist es, eine stilvolle, responsive und benutzerfreundliche Website zu erstellen, die die Atmosphäre eines Cafés vermittelt und Nutzern hilft, relevante Informationen schnell zu finden.

## ✨ Funktionalität

- **Responsive Landing Page:** optimierte Darstellung für Mobile, Tablet und Desktop.
- **Navigation zwischen Sektionen:** schneller Zugriff auf About us, Benefits, Gallery, Testimonial und Location.
- **Coffee Moments:** atmosphärische Bildergalerie, die Stil und Stimmung des Cafés zeigt.
- **Location:** Kontaktblock mit E-Mail, Telefonnummer, Adresse und Get-Directions-Link.
- **Benefits:** Darstellung der wichtigsten Vorteile des Cafés.
- **Testimonials:** Bereich mit Kundenbewertungen.
- **Subscription Form:** Formular mit Bestätigungsmodal.
- **Mobile Menu:** separate Navigation für mobile Geräte.
- **UI-Animationen:** Hover-Effekte, Animation des Browser-Tab-Titels und flüssige Interaktion mit UI-Elementen.

## Technologiestack

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Bundler:** [Vite](https://vitejs.dev/)
- **Deployment:** GitHub Pages
- **Bibliotheken und Tools:**
  - [Swiper](https://swiperjs.com/) — Slider und interaktive Blöcke
  - [body-scroll-lock](https://github.com/willmcpo/body-scroll-lock) — Steuerung des Scrollverhaltens bei geöffneten Menüs oder Modalfenstern
  - [skroll](https://www.npmjs.com/package/skroll) — Scroll-basierte Animationen
  - [gh-pages](https://www.npmjs.com/package/gh-pages) — Deployment auf GitHub Pages

## Installation und Start

### Voraussetzungen

- Installierte aktuelle LTS-Version von [Node.js](https://nodejs.org/).

### Schritte

1. **Repository klonen:**

   ```bash
   git clone <repository-url>
   ```

2. **Abhängigkeiten installieren:**

   ```bash
   npm install
   ```

3. **Projekt im Entwicklungsmodus starten:**

   ```bash
   npm run dev
   ```

4. **Öffnen Sie den Link, der im Terminal erscheint**  
   normalerweise `http://localhost:5173/`.

### Projekt bauen

```bash
npm run build
```

### Production-Build ansehen

```bash
npm run preview
```
