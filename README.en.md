## About the Project

**CoffeeJoy** is a team-built educational frontend project in the format of a coffee shop landing page. The website introduces users to the brand concept, café benefits, visual atmosphere, customer reviews, and location details.

Users can:

- Learn more about the coffee shop and its philosophy.
- Explore the main service benefits and visual gallery.
- Read customer testimonials.
- Find contact details and the café address.
- Subscribe to updates through a form.

The goal of the project is to create a stylish, responsive, and user-friendly website that communicates the atmosphere of a coffee shop and helps users quickly find the information they need.

## ✨ Features

- **Responsive landing page:** optimized layout for mobile, tablet, and desktop.
- **Section navigation:** quick access to About us, Benefits, Gallery, Testimonial, and Location sections.
- **Coffee Moments:** atmospheric image gallery showing the style and mood of the café.
- **Location:** contact block with email, phone number, address, and a Get Directions link.
- **Benefits:** presentation of the coffee shop’s key advantages.
- **Testimonials:** customer feedback section.
- **Subscription form:** form with a confirmation modal.
- **Mobile menu:** separate navigation for mobile devices.
- **UI animations:** hover effects, browser tab title animation, and smooth interaction with interface elements.

## Tech Stack

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Bundler:** [Vite](https://vitejs.dev/)
- **Deployment:** GitHub Pages
- **Libraries and tools:**
  - [Swiper](https://swiperjs.com/) — sliders and interactive blocks
  - [body-scroll-lock](https://github.com/willmcpo/body-scroll-lock) — scroll control for opened menus or modals
  - [skroll](https://www.npmjs.com/package/skroll) — scroll-based animations
  - [gh-pages](https://www.npmjs.com/package/gh-pages) — deployment to GitHub Pages

## Installation and Running

### Prerequisites

- Installed [Node.js](https://nodejs.org/) current LTS version.

### Steps

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the project in development mode:**

   ```bash
   npm run dev
   ```

4. **Open the link shown in the terminal**  
   usually `http://localhost:5173/`.

### Build the project

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
